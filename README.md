## Controls
| Key | Command |
| --- | --- |
| <kbd>A</kbd> <kbd>D</kbd> | Move |
| <kbd>Space</kbd> | Jump |
| <kbd>C</kbd> | Attack|
| <kbd>Esc</kbd> | Menu|
