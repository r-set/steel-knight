# Steel Knight
The Steel Knight in the catacombs. You need to pick up 2 keys to go to the next level,

## Controls
| Key | Command |
| --- | --- |
| <kbd>A</kbd>, <kbd>D</kbd> or <kbd>←</kbd>, <kbd>→</kbd> | Move |
| <kbd>Z</kbd>, <kbd>↑</kbd>, <kbd>Space</kbd> | Jump |
| <kbd>X</kbd>, <kbd>K</kbd>| Attack|
| <kbd>Esc</kbd> | Menu|
